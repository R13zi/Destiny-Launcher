Memoria alocada.
A Quantidade de Memoria alocada no modpack é um dos principais fatores que alteram o desempenho.
Lembre se de não alocar toda a sua memoria ram.

- RAM Máxima: Este valor indica a memoria RAM total utilizada no modpack, na qual o seu minecraft se limitará a utilizar durante sua execução (dica: caso tenha 8gb RAM + aloque entre 3-4 gb ram para ter um bom desempenho). Para computadores com menos memoria RAM, recomendamos utilizar 2gb de ram, e desabilitar a textura do servidor assim que logar.

- RAM Mínima : A quantia de memoria RAM minima, é o quanto seu jogo irá consumir no minimo, recomendo ficar entre 1 e 2 gb para computadores com 8gb RAM + e para computadores com menos, não passar de 1gb RAM.
